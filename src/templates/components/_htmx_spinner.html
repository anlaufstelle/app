{% comment %} Wiederverwendbarer HTMX-Lade-Spinner (Refs #1016, Workstream C — C3). Wird per ``hx-indicator="#"`` von den Live-Such-Eingaben referenziert. HTMX hängt während des Requests die Klasse ``htmx-request`` an dieses Element; die mitgelieferten htmx-Indicator-Styles (config.includeIndicatorStyles) blenden es dann via Opacity ein — kein eigenes CSS, kein Tailwind-Build-Eingriff nötig. Rein visuell: ``aria-hidden`` — die inhaltliche Änderung kündigt bereits die bestehende #flash-messages-Live-Region an (kein ARIA-Ausbau, A11y/M3-Embargo). SVG-Markup 1:1 aus statistics/partials/dashboard_content.html. Parameter: spinner_id Pflicht — DOM-id, auf die hx-indicator zeigt. position_class Optional — Positionierungs-Utilities (z.B. absolute-Layout). {% endcomment %}
